# RACI

## 1. 目的

本書は、SpecDojo Handbook プロジェクトにおける成果物作成、レビュー、承認、変更管理の責任分担を明確にするため、RACI を定義する。

## 2. RACI の定義

| 記号 | 意味        | 説明                     |
| ---- | ----------- | ------------------------ |
| R    | Responsible | 実作業を担当する         |
| A    | Accountable | 最終責任を持ち、承認する |
| C    | Consulted   | 相談・レビューに参加する |
| I    | Informed    | 結果の共有を受ける       |

| 成果物                     | PO  | PM  | BA  | ARC | QE  |
| -------------------------- | --- | --- | --- | --- | --- |
