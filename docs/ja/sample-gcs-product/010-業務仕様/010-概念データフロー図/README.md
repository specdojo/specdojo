# 概念データフロー図

業務領域の全体的な構造を、概念レベルのデータフローにより可視化する。

## ドキュメントの目的

業務（プロセス）で扱う情報や物の流れ、業務のトリガーとなるイベント、担当者を可視化し、対象業務領域の俯瞰的な理解を促進する。
