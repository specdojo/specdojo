# C4コンテナ図

システム境界内の主要コンテナ（アプリケーション/データストア等）と、関係者・外部システムとの関係を、C4コンテナ図として定義します。

## 記法ガイド

1. 図を含むドキュメント（Markdown）の作成・更新ルールは [cnd-rulebook.md](../../../specdojo/rulebooks/cnd-rulebook.md) を参照してください。
2. Mermaid 図の作図ルールは [cnd-mermaid-rulebook.md](../../../specdojo/rulebooks/cnd-mermaid-rulebook.md) を参照してください。
